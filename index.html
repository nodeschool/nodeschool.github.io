LEARN NODE.JS WITH INTERACTIVE LESSONS
Learn You The Node.js For Much Win!
Learn the basics of node: asynchronous i/o, http
Once you have node installed, run: 
sudo npm install -g learnyounode
Now you have a learnyounode command. Just run: 
learnyounode
to play!
Stream Adventure
Learn how to compose streaming interfaces with .pipe()
Once you have node installed, run: 
npm install -g stream-adventure
Now you have a stream-adventure command. Just run: 
stream-adventure
to play!
Level Me Up Scotty!
Learn to use leveldb, a simple key/value store with a vibrant package ecosystem.
Once you have node installed, run: 
npm install -g levelmeup
Now you have a levelmeup command. Just run: 
levelmeup
to play!
